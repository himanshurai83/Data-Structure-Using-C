# 🧠 Data Structures in C

This repository contains implementations of fundamental **Data Structures in C language** with all major operations.
It is designed to strengthen core programming concepts and build a strong foundation for problem-solving and interviews.

---

## 📌 Topics Covered

### 🔢 Array Operations

* Insertion (Beginning, End, Specific Position)
* Deletion (Beginning, End, Specific Position)
* Traversal
* Searching (Linear Search)
* Updating elements

---

### 🔗 Linked List Operations

* Singly Linked List implementation
* Insertion:

  * At Beginning
  * At End
  * At Position
* Deletion:

  * From Beginning
  * From End
  * From Position
* Traversal
* Searching

---

### 📚 Stack Operations (LIFO)

* Push
* Pop
* Peek (Top element)
* Check if empty

---

### 🚶 Queue Operations (FIFO)

* Enqueue
* Dequeue
* Front / Rear operations
* Check if empty

---
